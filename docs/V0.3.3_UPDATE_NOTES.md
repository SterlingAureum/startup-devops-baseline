## Design choice

This version uses a minimal canary analysis:

```promql
sum(up{job="demo-api-canary"})
```

This verifies whether Prometheus can scrape the canary service during rollout.

It does not yet measure real application success rate because the current demo-api metrics do not expose HTTP status-code labels.
