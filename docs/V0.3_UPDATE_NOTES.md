# v0.3 Update Notes

This package adds Progressive Delivery support.

## Added

- Argo Rollouts platform Application.
- demo-api Rollout template.
- Rollout-enabled Helm values.
- Rollout helper scripts.
- Progressive delivery documentation.
- Rollback runbook.

## Updated

- demo-api Application adds sync wave and `SkipDryRunOnMissingResource=true`.
- GitHub Actions now verifies that Helm renders a Rollout resource.

## Apply Order

1. Copy files into the repository.
2. Commit and push to the repository watched by Argo CD.
3. Sync `startup-devops-root`.
4. Confirm `argo-rollouts` is healthy.
5. Confirm `demo-api` renders as a Rollout.

## Validation

```bash
kubectl get application -n argocd
kubectl get pods -n argo-rollouts
kubectl get rollout -n startup-apps
kubectl argo rollouts get rollout demo-api -n startup-apps
```
